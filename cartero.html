Cartero

Cartero is a discord bot. The tagline is- Helping servers grow exponentially with a simple solution. Make your server active and entertain your community with a simple solution - a single discord bot!

This bot allows basic server moderation that includes purge messages, mute, unmute, kick, ban, unban. But, bots are meant to be fun, so this bot lets you play blackjack with a friend of yours, or if you don't have one, you can play with the bot. You can beg, bet, and transfer the bot's currency, also view your and your friend's blackjack records.

This bot is programmed using discord.py library. It uses JSON to store user and server data. The bot is hosted using Amazon Web Services. With time, everything gets better, so will the bot.
